How to install on my phone?

Letter Jam Virtual Sheet

  • Tap a cell to enter a single letter (A–Z). Leave blank if unknown.
  • Use Add/Remove Row to match your word length or clue rows.
  • Save stores everything on your device. No internet needed.
  • Export creates a JSON file; Import loads one back.
  • Use the Install button (or browser menu) to add TableTop HQ to your home screen.
